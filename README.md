# 日報も、ふりかえりも、職務経歴書も。書く時間を減らす無料ツールを作りました 🐰

毎日の日報、月末の工数まとめ、たまの職務経歴書（スキルシート）の更新。
「やったほうがいい」とわかっているのに、書くのに時間がかかって後回しにしがちですよね。

そこで、**書く時間をできるだけ短くするための道具**を作りました。
登録もインストールも不要。ブラウザでファイルを開くだけで使えます。

この記事では、なぜ作ったのか・誰のためのものか・何が解決できるのか・どう使うのかを、順番に書きます。

---

## なぜ作ったのか（きっかけ）

きっかけは、まわりで「記録が続かない」「書くのに時間がかかる」という声をよく聞いたことでした。

日報は毎日のこと。なのに、フォーマットを思い出し、言葉を選び、コピペ先を整える。
その一つひとつが地味に時間を奪っていきます。
さらに月末になると「今月、何にどれだけ時間を使ったんだっけ」と思い出せず、工数報告に苦労する。
職務経歴書を書こうとすると、毎日の作業を覚えていなくて手が止まる。

**毎日の小さな記録が、月末のまとめにも、経歴の棚卸しにもつながっていない。**
それが、いちばんもったいないと感じていました。

---

## なぜ大切なのか（Why）

書くこと自体が目的ではありません。大切なのは、その先にあるものです。

- **ふりかえりは、成長の入り口**。今日うまくいったこと（Keep）を1つ言葉にするだけで、明日が少し良くなります。
- **記録は、自分を守る資産**。「何をやってきたか」が残っていれば、評価の場でも転職の場でも、自分の言葉で語れます。
- **時間の見える化は、働き方の調整につながる**。何にどれだけ使っているかが見えて初めて、減らす・増やすの判断ができます。

だからこそ、書くための負担は小さいほどいい。
**考える時間と手を動かす時間を増やすために、書く時間を減らす。** それがこのツールの目的です。

---

## 誰のためのものか（対象）

次のような方を想定しています。

- 毎日**日報や進捗報告**を書く必要がある方
- **月末に工数や稼働をまとめて報告**する立場の方
- **職務経歴書（スキルシート）**を継続的に更新したい方
- ひとりで**ふりかえり（KPTやPDCA）**を回したい方
- 記録は**自分の端末の中だけ**に置いておきたい方

ITの研修講師やエンジニアの現場を想定して作っていますが、毎日の記録を扱う方なら職種を問わず使えます。

---

## 何が解決できるのか（できること）

ひとつのツールに、用途別の画面が入っています。重複なく、役割で分けて整理すると次の通りです。

### 1. 日々の記録を、すぐ形にする

- **業務日報**：決まった項目に沿って、退勤前の報告がさっと書けます
- **メモ版 日報**：チェックリスト形式で、忙しい日でも軽く残せます
- **相談チャットテンプレ**：Slackなどに貼れる相談・質問・進捗報告の文面が作れます

### 2. 時間を、見えるようにする

- **時間割**：1日をコマで区切り、予定・実績・達成度を記録。時間を自動集計します
- 丸め単位を **5分・10分・15分・30分・1時間** から選べ、`#タグ` で分類もできます
- 急な割り込み対応も記録でき、「予定外にどれだけ時間を取られたか」が見えます

### 3. ふりかえって、次につなげる

- **一人MTG（PDCA）**：ひとりでPDCAを回す記入欄＋タイマー
- **一人MTG2（目標・KPT）**：最低限と最高、2段階の目標とふりかえり

### 4. 月末と経歴に、地続きでつなげる

- **月次データ統合**：1か月分のメモを取り込んでまとめ、月末報告書にもスキルシートにも送れます
- **スキルシート（案件欄）**：職務経歴書の案件欄を、サイズ感をガイドしながら作成
- 月末報告書はExcelに出力でき、タスク別・日別・タグ別の工数表が手に入ります

### 5. 自分の型で書く

- **自由ひな形**：自分の日報フォーマットを登録して、差し込み式で生成できます

## どう使うのか（使い方）

使い方はシンプルです。

1. ファイルをダウンロードして、**ダブルクリックでブラウザで開く**
2. 上のタブから目的の画面を選ぶ
3. フォームを埋めて「生成する」ボタンを押す
4. 画面下に出てきた文章を、**コピー／テキスト保存／マークダウン保存**で使う

入力した内容は自動で保存されるので、続きはまた次の日に。
日本語と英語の切り替えにも対応しています（右上のボタン）。

> 💡 大事なデータは、Excelやテキストで書き出してバックアップしておくと安心です。
> 入力内容はあなたの端末の中だけに保存され、どこにも送信されません。
